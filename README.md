# Marie-Assembly-Mini-Projects
Couple of MARIE programs showing basic programming concepts

## Features
- Draw shapes on a memory-mapped display based on user input
- Use subroutines to control program flow
- Compare assembly language and machine language implementations
- Visual output using MARIE display memory

## File Overview
- 2.3_Display4Directions.mas: Program to draw shapes (square, rectangle, triangle) based on user input
- 2.4_AssemblyANDMachine.mas: Demonstrates the difference between assembly and machine language
